01Getting in

Open the app at cbdcollege-scheduler.pages.dev. The first screen is the sign-in form.

Sign in

1Type your email address.
2Type your password.
3Tap Sign In.

If the password is wrong, a red banner appears beneath the form explaining the error. Trainer/admin accounts are created from the Admin tab once you're inside.

Sign out

Top-right of the header: Sign Out. Clears the session and returns to the login screen.

Theme — Dark or Light

The 🌙 / ☀️ button in the header toggles dark mode. New visitors see dark mode by default; your preference is saved and survives sign-outs.

Install as an app (mobile)

On a phone, the 📱 Install button appears in the header until you've installed the PWA. Tap it and follow the system prompt — on iPhone you'll see step-by-step instructions for using Add to Home Screen.

Enable mobile push notifications

Tap 🔔 Enable Push in the header. Grant the system prompt. After that, you'll receive a real OS notification (lock screen / banner) for every roster change, even when the app is closed.

iPhone only On iOS, push notifications only work after you've installed the PWA to your home screen. Tap Install first, open the installed app from the home screen, then tap Enable Push.

02Reading the app

The header

Top blue bar — always present. Contains:

  • Your name and role badge (TRAINER)
  • View as Assistant — preview what assistants see
  • 📱 Install — install the PWA (mobile only, hides once installed)
  • 🔔 — notification bell with unread badge
  • Export PDF — print a roster PDF
  • 🔔 Enable Push — opt into mobile pushes (hides after granted)
  • 🌙 / ☀️ — theme toggle
  • Sign Out

The main tabs

Just below the header:

Schedule Classes Staff Notifications Admin

The Schedule and Staff tabs each contain two sub-views, shown as pill buttons under the main tab strip when you click them:

  • Schedule → Per-staff + Team
  • Staff → Manage Staff + Availability

Calendar indicators

Every calendar in the app uses the same visual language for each day cell:

✓ Rostered / class configured ✕ Class day but not rostered / not configured Today (blue ring + TODAY pill) Selected for multi-edit

On desktop, cells show the day number plus rich detail — class times, student counts, staff initials, role badges. On mobile, cells collapse to just the day number plus a green check or red cross to keep everything readable on a small screen. Tap any cell to open the full detail modal.

03Schedule — Per-staff view

This is where you manually roster a specific staff member on or off individual days.

Pick the staff member

Use the Staff dropdown at the top of the calendar to choose whose roster you're editing. The calendar refreshes to show their availability for the current month.

Set availability for a single day

1Tap a day cell.
2The day editor modal opens. Choose a status: Available, Partial, or Unavailable.
3Optional: set a custom start/end time, role (Assistant / Head Trainer), and notes.
4Tap Save.

The cell turns green (available), amber (partial), or red (unavailable). Affected staff get a push notification titled "Your roster was updated".

Remove a day

Open the same day modal and tap Remove. The day reverts to the unset state and the staff member is notified.

Roster many days at once

1Tap day cells one by one — selected cells get a blue outline.
2A floating toolbar appears at the bottom with bulk actions.
3Choose Available, Partial, or Unavailable — or Clear to wipe entries.

Clear all days in this month

The Clear month button on the calendar controls removes every roster entry for the selected staff in the current month. Confirms before acting.

Today / month navigation

Use ‹ and › to step months; Today jumps straight to the current month.

Today / Next Shift hero card

Above the calendar, the blue hero card summarises today's class (times, student counts, who's rostered). It refreshes automatically as you make changes.

Import a roster from Excel

The Import Roster button opens the Excel/CSV import wizard. See Section 10 for the full walkthrough.

Auto-roster

The Auto-roster button generates a roster from saved availability + priorities. See Section 11.

04Schedule — Team view

A read-only overview showing who's rostered across every day in the month.

Reading a cell

Each desktop cell lists up to three rostered staff by first name and colour dot. Head Trainers show a HT badge. Student counts (AM / PM) appear beneath; AM CAP / PM CAP badges mark sessions at capacity.

Open day detail

Tap any day to open the day-detail modal — full list of rostered staff, class times, AM/PM student counts, and cap status.

05Classes — Default class times

Set the default AM and PM class times, student counts, and capped status for each day of the week. These defaults are applied to every date unless that specific date has been manually overridden.

Edit defaults for a single day

1Find the row for the day (Mon, Tue, …).
2Set AM Class start and end.
3Set PM Class start and end.
4Type the expected student counts for AM and PM.
5Tick Cap AM / Cap PM if either session is at capacity (limits auto-roster).
6Tap Save Defaults at the bottom.
Mobile note Each day becomes a stacked card with fields labelled — no horizontal scrolling needed.

Save vs Save & Notify

After tapping Save Defaults a confirmation appears. You get two choices:

  • Save only — writes the change quietly.
  • Save & Notify — writes the change and pushes an OS-level notification to every signed-in staff member.

Quick-copy shortcuts

  • Copy Mon to Tue–Fri — mirrors Monday's settings across the weekdays.
  • Copy Sat to Sun — mirrors Saturday's settings to Sunday.
  • Copy Mon to all days — overwrites every day with Monday's settings (useful for uniform schedules).

06Staff — Manage staff members

Add a staff member

1Tap + Add Staff in the Staff Members card.
2Type their full name.
3Pick a colour (used for their dot/chip everywhere).
4Tick Head Trainer if they are one. HTs are excluded from the assistant priority list and are auto-rostered based on their own availability + weekend rotation.
5Set their priority (lower = picked first in auto-roster). Leave blank to put them at the end.
6Optional: link this staff record to an existing user account so they receive notifications and can see their own roster.
7Tap Save.

Rename a staff member / edit details

1On their card, tap the pencil icon.
2Edit name, colour, role, priority, or linked user as needed.
3Tap Save.

Remove a staff member

1On their card, tap the trash icon.
2Confirm the deletion. This also removes their roster entries — it cannot be undone from inside the app.
Heads up Removing a staff member deletes their availability history along with the staff record. If you just want to stop them being auto-rostered, set their priority very high or leave them unavailable instead.

Link a staff record to a user account

If a staff member has a sign-in account (so they can see My Roster), edit their staff entry and pick their user in the Linked user dropdown. Until linked, they won't receive in-app or push notifications.

07Staff — priority, slot times, rules

Priority Order by Day

Lower number = picked first when auto-rostering. Head Trainers are excluded — they're rostered based on their own availability.

1Pick a day (Mon, Tue, …) at the top of the Priority Order card.
2Drag assistants up/down to set the order, or type a number into each row.
3Tick Rotate priority each [day] if you want the top spot to alternate between two assistants each week.
4Tap Save This Day.

Two helpers to apply the current order quickly:

  • Apply to Tue–Fri
  • Apply to all days

Availability sub-tab

Click Staff in the top nav, then choose Availability from the pill bar that appears underneath. This view lets trainers override any individual assistant's availability for any date.

1Pick the assistant.
2Tap a day in their calendar.
3Set Available or Not Available, add an optional note.
4Tap Save. The assistant receives a targeted push titled "Your availability was updated".
5To remove an override, open the same day and tap Clear.

Assistant Start Times by Slot

Once an assistant is auto-rostered into a class, their start time is determined by which "slot" they were picked into — Slot 1 = first picked, Slot 2 = second, etc. The grid lets you set the start time per slot per AM-class group.

Edit any value and tap Save Start Times. The same Save / Save & Notify dialog appears.

Auto-roster Rules

Two settings live here:

  • Need 1 assistant when class ≥ N — single-assistant threshold.
  • Need 2 assistants when class ≥ N — second-assistant threshold (must be ≥ the first).
  • Who works the first Saturday of each month? — picks which Head Trainer kicks off the weekend rotation; the other HT works Sunday, and they swap each subsequent weekend.

Tap Save Rules. Same Save / Save & Notify confirmation.

08Notifications tab

Compose and send arbitrary notifications to one staff member or every signed-in user.

Send a notification

1Tap Notifications in the top tab strip.
2Pick a recipient from the dropdown — All staff (broadcast) or a specific staff member.
3Type a title (up to 120 chars) — e.g. "New roster posted".
4Type the message body (up to 400 chars).
5Tap Send Notification.
6Confirm the preview, then tap Send.
What happens The recipient(s) see the notification in the in-app bell instantly, and any subscribed devices buzz with a system push.

Reading what you've received

The 🔔 bell in the header opens the in-app notification panel — list of recent notifications targeted to you or sent as broadcasts. Tap an item to mark it read.

09Admin tab

Create, edit, and manage user accounts.

Create a user account

1Tap + Create User.
2Enter their email, a temporary password, and pick a role (Trainer / Assistant).
3Submit. The new row appears in the table; share the credentials with the user securely.

Edit a user

Tap the pencil icon in the Actions column to change role or rename. Tap the trash icon to delete the account.

After creating a user Open the Staff tab and link the new user account to the matching staff record so they get personal roster notifications.

10Import a roster from Excel

Bulk-load a month's worth of class info from an Excel/CSV file produced by your existing roster process.

Open the import wizard

In the Schedule tab, tap Import Roster. A modal opens.

Steps

1Tap Choose file and pick the .xlsx or .csv export.
2The wizard parses the file and shows a preview — one row per detected day, with the matched Head Trainer, AM/PM students, and a tick / warning per day.
3Review warnings. Yellow flags indicate something the parser couldn't match (e.g. an unknown trainer name).
4Untick any days you don't want to apply.
5Tap Apply Import. Confirms once more, then writes the days to the database.
Format requirements The file must have a 7-day grid in the layout used by the existing roster spreadsheet — rows in groups of 5 with labels ABS 3 hrs am, ABS 2 hrs Art, Trainer 2, Total # of students in the first column.

11Auto-roster from availability

Generate a draft roster for the visible month based on each staff member's saved availability + priority order + the auto-roster rules.

Run auto-roster

1Schedule tab → tap Auto-roster.
2The modal shows a preview of every day that would change — additions in green, removals in red.
3Review the changes. Untick any days you don't want to apply.
4Tap Apply.

Auto-roster respects:

  • Per-day priority order and rotation flag (Staff tab).
  • Class size thresholds (Auto-roster Rules card).
  • Capped sessions — no assistants added to capped AM/PM.
  • Weekend rotation for Head Trainers.
  • Each staff's Available entries — no one is rostered on a day they marked unavailable.

12Exporting a PDF roster

Steps

1Tap Export PDF in the header.
2Pick the month range (defaults to the currently visible month).
3Pick which staff to include (default: all).
4Tap Export. The browser downloads the PDF.

The PDF lays out one section per month with a per-day grid showing rostered staff, times, and capacities. Suitable for printing or emailing.

13How notifications work

Two layers

  1. In-app bell — the 🔔 icon in the header shows a red unread-count badge. Click to open the recent notifications panel.
  2. Mobile push — if the user has tapped 🔔 Enable Push and granted permission, their device receives a real OS-level notification (lock screen / banner / sound).

What triggers a notification

TriggerRecipient
Trainer changes a staff's roster for any dayThat staff member (targeted push)
Trainer overrides an assistant's availabilityThat assistant (targeted push)
Default Class Times saved with "Save & Notify"All staff (broadcast)
Auto-roster Rules saved with "Save & Notify"All staff (broadcast)
Slot Times saved with "Save & Notify"All staff (broadcast)
Notifications tab — manual sendChosen recipient (one or all)

If a push doesn't arrive

  • Confirm the receiver has tapped 🔔 Enable Push and granted the OS prompt.
  • On iPhone: the app must be installed to home screen (PWA), not viewed in Safari.
  • Check the recipient's account is linked to a staff record (Edit Staff → Linked user).
  • If the browser is blocking notifications, the user must un-block in browser settings before re-enabling.

14View as assistant

Useful for verifying what an assistant actually sees in their account before posting a roster.

Steps

1Tap View as Assistant in the header.
2The tab strip switches to My Roster / My Availability, and the preview bar offers a staff selector so you can choose whose view you're previewing.
3Pick a staff member; the calendar shows exactly what they'd see when signed in.
4Tap Back to Trainer view to leave preview mode.

15Troubleshooting

An assistant says they didn't get notified about a change

  • Confirm their staff record has a Linked user.
  • Confirm they've tapped 🔔 Enable Push on their phone.
  • If iPhone, confirm they've installed the PWA via the Install button.
  • The in-app bell always works regardless of push status — ask them to check the 🔔 icon.

The Schedule calendar shows blank cells

  • You probably haven't picked a staff member in the dropdown — it defaults to your own account, but a fresh install may have no selection.
  • If the staff member exists but has no availability records yet, every cell will be empty until you set entries (manually or by import / auto-roster).

Defaults aren't being applied to a specific date

A date with manually-set class times wins over the day-of-week default. To revert, open that day's class edit modal and clear the times so it falls back to the default.

Push notifications fire twice on the same device

That happens if a stale push subscription is still alive. The next time the device receives a push it'll be cleaned up automatically — no action needed.

Mobile push isn't working on iPhone

iOS requires the app to be installed to the home screen before push will deliver. Tap Install, follow the system prompt, then open the installed app from the home screen and tap 🔔 Enable Push.
